# Commissioning Checklist (Template)

## Facilities
- [ ] Power verified (voltage, current capacity)
- [ ] Network ports / VLAN / IP addressing approved
- [ ] Utilities verified (CDA/exhaust/cooling)
- [ ] Permits + inspections complete (if needed)

## Safety
- [ ] E-stop functional test
- [ ] Door/light curtain interlocks verified
- [ ] Emergency vent / pressure relief (if applicable)
- [ ] LOTO plan accessible + reviewed

## Controls
- [ ] PLC/firmware version matches release lock
- [ ] HMI build matches release lock
- [ ] Alarm taxonomy loaded + verified

## Hardware
- [ ] Wiring continuity passed
- [ ] Actuators homed safely
- [ ] Calibration complete (as needed)
