# Bring-up Plan (Commissioning)

## 0) Preconditions (no exceptions)
- EHS: training complete, PPE, LOTO plan reviewed
- Facilities: utilities verified, permits/signoff as required
- Controls: release lock established (versions pinned)
- Hardware: wiring continuity check complete

## 1) Install + utilities
- Verify power, grounding, network, CDA/exhaust/chilled water (as applicable)
- Record as-built deviations

## 2) Power-on
- Perform safe power-on checklist
- Verify safety chain and interlocks

## 3) IO validation
- Validate every IO point against IO map
- Log mismatches as blocking issues

## 4) Subsystem checks
- Motion/actuation (low-speed)
- Sensors calibration
- Alarm taxonomy sanity check

## 5) First integrated run
- Minimal recipe / low-risk settings
- Capture logs + baseline metrics

## 6) Exit criteria
- No Sev-1 safety issues
- IO map matches as-built
- Key subsystems run repeatably
- Open issues have owners + dates + containment
